# YBC 后台管理系统

## 项目简介

一个基于 Vue.js 和 Element Plus 的前端应用

## 技术栈

- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Element Plus
- Tailwind CSS
- Axios
- MD5 加密算法

## 安装与运行

1. 克隆项目：

```bash
git clone <你的项目地址>
```

2. 进入项目目录：

```bash
cd <项目目录>
```

3. 安装依赖：

```bash
npm install
```

4. 启动开发服务器：

```bash
npm run serve
```

在浏览器中访问 http://localhost:8080 进行访问。
